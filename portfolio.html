Cake maker project


The program lets the user input values that lets you custamize a cake by varying the flavors of layers of the cake, the size of the cake, the amount of icing, and the amount of candles you want on your cake. Resulting in what ever cake you desired.


1.2.5 Hangman Project


The program lets you play hangman. The user guesses a letter and if it's correct the letter is added to it's corresponding dash. If it's wrong the program starts to draw the man being hanged. The user is only given seven incorrect guesses before the man is hanged and the user losses the game.
